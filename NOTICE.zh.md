# 版权与项目关系说明

本项目是由社区独立开发的文件格式研究与地图编辑工具，与《三国霸业》的开发商、发行商及其他权利方没有隶属、授权或背书关系。

源码仓库和基础 Windows ZIP 不提供原始游戏程序、地图、头像、文本、关卡文件或预先生成的素材图片。基础 ZIP 只含编辑器程序、无资源入口、发布元数据和文档，并通过显式白名单与禁用资源扫描。

用户必须自行准备合法取得的游戏副本并遵守所在地法律及原软件许可。原始文件导入只用于本机系统临时会话；用户也可生成在本机持久缓存中复用的 `.sanmap-pack` 独立内容包。

独立内容包和含 `content-packs/` 的组合 ZIP 包含用户生成的游戏数据、派生图片与写回参考文件，不属于基础无资源发布包。创建者和分发者必须自行确认其保存、使用与分发权利，不能用基础包已经通过资源审计替代该确认。
